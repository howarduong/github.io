This document describes how to use Maya to define portals and create hard points. It is not meant as a substitute to the lessons Create Low-Polygon Characters and Create an Internal Shell – those documents are essential reading.

This lesson assumes that the BigWorld Maya Exporter and the BigWorld shelf have already been installed on your machine. For details on how to install them, see Content Tools Reference Guide, chapter 3ds Max and Maya Exporter, section Maya → Installing the 3ds Max Exporters.

Using the exporter

1 Start the export process.

Select the File→Export All menu item to export all objects in the scene – this will export all objects, even the hidden ones.

–or–

Select the File→Export Selection menu item – this will export only currently selected object.

The Export dialog box will be displayed.

3 In the Export dialog box:

A In the Look In drop-down list box, specify the folder where the asset should be saved.

B In the File Name drop-down list box, specify the name of your asset.

C In the Files Of Type drop-down list box, select the BigWorld Asset item.

D Click the Options button – depending on the menu item previously selected, the Export All Options or the Export Selection Options dialog box will be displayed.

In the dialog box's General Options group box, set the File Type drop-down list box to BigWorld Asset.

Click the Export All or Export Selection button (the button displayed depends on the menu item previously selected) – this will close the current dialog box and return to the Export dialog box.

E Click the Export button – the Visual Exporter dialog box will be displayed.

In the Visual Exporter dialog box, click the OK button – the asset will be exported to the specified folder.

Defining portals using BigWorld shelf

The BigWorld Maya shelf allows users to define objects as portals. It contains 3 icons, which allow you to define objects as Portal, Exit, or Heaven portal types.

The shelf contains the following buttons:

– Add Portal flag

– Add Exit flag

– Add Heaven flag

To assign a mesh object as a portal type, first select it, then click the portal type from that you wish to assign it. You can check your assignment by viewing the attribute editor.

The Extra Attributes rollout contains the Portal check box – it will contain a Heaven check box for Heaven portals, and an Exit check box for Exit portals.

After the portals have been assigned the shell is ready for export.

Creating hard points within Maya

This document is not meant as a substitute to the lesson Create Low-Polygon Characters, section Placing hard points on your character for attaching items – please read that section before continuing.

Hard points should be created as a polygon object, and represent the directions X, Y, Z of the object's node. This makes it easy to position the hard points in the appropriate place.

The shape, texture and geometry are only visual cues for the artist. The BigWorld engine requires only for the hard point to use the HP_ naming convention, and for it to be a node of the object that it is attached to.

For the purposes of this tutorial, we will be creating a hard point that will be positioned in the right hand of a character avatar.

1 Create and position a a suitable piece of geometry.

2 Select the Window → Hypergraph menu item – the Hypergraph window will be displayed.

3 Right-click the node, then select the Rename option.

4 Rename the node HP_RightHand.

5 Middle-click the HP_RightHand node, then drag it onto the parent RightHand node.

6 The hard point is now attached and ready to use.
