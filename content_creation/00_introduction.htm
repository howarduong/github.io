Introduction

Welcome to the artist training tutorials for BigWorld technology.

The following lessons are designed to be progressive, meaning that you should follow them from start to finish in order to learn the complete production workflow.

Although 3ds Max is used in many of the examples in this manual, the same themes and concepts apply for Maya users. Hence, it is inadvisable to skip tutorials.

Content Tools Reference Guide–Further information about the tools can be found the Content Tools Reference Guide.

3ds Max example files are also provided to assist in learning the processes.
Example files can be found in bigworld/doc/tools_tutorial_examples/. This tutorial folder contains the following:

A Folder bigworld/doc/tools_tutorial_examples/working_files, under which you will find 3ds Max files relating to the tutorials found in this guide, and their associated maps, laid out in a folder structure similar to the one recommended in the lesson Manage Resources.

An empty file structure which you may chose to use to start your project. If you copy this into your mf/your_game folder, then you will be able to use the empty /res folder as a place to put your art assets if you wish.

A BigWorld Material library, for 3dsMax which contains all the BigWorld shaders.

The following tools are used throughout the tutorials:

ModelEditor - To view or modify your exported objects.

WorldEditor - To build the game world.

ParticleEditor - To create and modify particle systems.

3ds Max - To create 3D content, including the proprietary BigWorld exporters.

Maya - To create 3D content, including the proprietary BigWorld exporters.

Exporters - To export objects and animations from your chosen 3D application. Instructions on how to install the BigWorld exporters are found in the Content Tools Reference Guide.

Photoshop - To create texture maps.

SpeedTree - To create dynamic trees.
