Changing the surface height

To change the height of a surface, follow the steps below:

1 WorldEditor uses a 3D brush to sculpt the terrain.

To access it, click the toolbar button – this activates the Terrain Height panel. The Size and Strength fields in the panel determine the magnitude and speed of the change.

Terrain Height panel

The limits of each slider can be set by pressing the button next to it (this will invoke the Set The Slider Limits dialog box.

Dialog box used to set the limits of a slider

2 The brush will be displayed in the viewport as a 3d dashed ring that defines the area of influence.

Brush cursor in the viewport

3 Drag the Size slider, or use the shortcut keys [ (left square bracket) and ] (right square bracket) to alter the brush diameter, thereby changing the influence area.

The Strength field can be increased or decreased via keyboard shortcuts ALT+[ and Alt+] respectively. It will not cause any visual changes to the brush icon, only to the speed at which it affects the underlying terrain.

4 To apply the brush, simply hold down the left mouse button and drag over the desired area.

The terrain will rise up in the center of the cursor. If you do not notice any changes in the height of the terrain, then increase the value of the Strength field.

The Brush Falloff group box determines the shape of the terrain around the outer edge of the brush, with three options provided – Linear, Curve, Flat and Average.

Terrain raised using the three options Linear, Curve, and Flat brush

You can undo the brush operation at any time, via the toolbar button or the Ctrl+Z keyboard shortcut, both of which can be applied multiple times.
Note that one undo operation is based on a single mouse click. This means that if you apply a brush operation by holding down the mouse button, and then undo it, the result of the entire change will be removed.
To avoid losing too much progress, it is recommended to paint in shorter dabs, instead of using longer brush stroke periods.

5 To lower the terrain, drag over the desired area with the middle mouse button.

Terrain lowered under brush cursor

6 The Left Shift key can be used to toggle between Filter and height mode.

7 Using the Explicit Height field, you can snap the terrain to a specified height in metres.

To set an explicit height for a particular area of the terrain, follow the steps below:

A Set the value of Explicit Height field to the desired value (in metres) – the terrain height can be sampled using the Alt key.

B While pressing Ctrl, drag over the desired area – the terrain under the brush cursor jumps to the specified height.

Explicit Height field

Terrain height raised to explicit value

You must click and drag the cursor over the desired area to change its height – just clicking while holding Ctrl only changes the height of the point clicked.

The Explicit Height value can be added in two ways: Absolute (where terrain height is set to the value of Explicit Height) and Relative (where value of Explicit Height is added to the height of terrain).

8 There are three ways to visualise the terrain mesh:

A by selecting General Options panel's Show list box's Terrain→Wireframe entry;

Terrain→Wireframe option in General Options panel

B by clicking one of the Chunk Visualisation toolbar buttons: (Normal mode), (Chunk borders mode), (Pole markers mode), or (Wireframe mode); or

C by pressing F6 keyboard shortcut to cycle through the Chunk Visualisation modes (in the order in which they appear in the toolbar).

Average brush

The Average brush is great for creating quick paths and roads. It works by taking a sample of all the height positions under the cursor and then averages them.

To use the average brush;

A click the toolbar button to activate the Terrain Height panel;

B Select the Average Radio button

C Adjust the Size and Strength

Here you can see an examples of what the average brush can achieve.

Adding refinement filters

To refine the terrain using filters, click the toolbar button or use the shortcut key "4" – this activates the Terrain Filtering panel.

1 Filters apply an algorithm to the area you paint. That algorithm can smooth, sharpen or add noise to the terrain.

Terrain Filtering panel

Smooth filters are ideal for ironing out any imperfections caused by manually brushing with the height tool.
Sharpen filters are ideal for increasing small differences in the terrain.
Noise filters can break up flat areas and give the terrain a more natural appearance.

2 Apply the desired filter.

To use filters on the terrain, follow the steps below:

A click the toolbar button to activate the Terrain Height panel;

B adjust the Size, Strength, and Explicit Height fields to the desired values;

C select the desired Brush Falloff;

D click the toolbar button to activate the Terrain Filtering panel;

E select the desired filter in the Filters list box;

F apply the selected filter in the viewport, using the click-and-drag method.

Filter effects

Removing terrain

The toolbar button activates the Terrain Mesh Cut/Repair panel, which gives you access to the cut and repair actions, allowing you to remove or replace entire sections of terrain.

Terrain Mesh Cut/Repair panel

To remove the terrain, follow the steps below:

1 select the Cut option button;

2 set the Size field to the desired value – this will set the size of the brush, which is represented as a green square;

3 navigate the camera to an overhead view, in any of the following ways:

use the movement keyboard shortcuts Q, W, E, A, S, and D.

drag the mouse to change the camera angle;

click the toolbar button to turn Orthographic View on.

4 Cut the terrain:

click the mouse to cut terrain of the specified size; or

click and drag the mouse over the desired area to cut it – the terrain will be cut as you move the mouse over it.

Terrain removed with Cut action

5 The Repair action replaces the terrain that has been removed, based on the size specified for the brush. The Left Shift key will toggle between cut and repair mode

Terrain replaced with Repair action
