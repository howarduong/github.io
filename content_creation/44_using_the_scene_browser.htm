The Scene Browswer tab in World Editor is a spread sheet view of all the objects loaded in a space

Scene Browswer

The Scene Brower has many possible applications some of which are listed below.

 

Finding objects – The search field will filter objects by name, right clicking on an object allows you to zoom to the objects location.

Selecting multiple objects – multiple objects can easily be selected and their common properties edited at once

Modifiying multiple objects –When multiple objects of the same type are slected their common properties can be modified from the properties tab

Monitoring Triangle count per assets – The column Tris can be used to sort assets by the number of triangles, very useful when looking for polygon heavy assets.

Monitoring Primitive groups per assets – The column Primitives gives an indication of how many parts each asset is broken up into. This is usually due to multiple material kinds or high bone counts. Large primitive group numbers will impact performance, it is best to investigate assets at the top range to see if their primitive group count can be reduced. It's rare that objects should need greater than 10 primitive groups.

Hide / Show assets – The scene browser can be used to hide assets to enable easier world building. Hidden assets are only hidden in World Editor, they will be drawn as usual in the client engine.

Freeze / Thaw assets – The scene browser can be used to freeze and thaw assets to enable easier world building. Frozen assets will appear grey and be unselectable from the 3d window. If you need to unfreeze an asset, select it from the scene browser and click the thaw button.

 

Common properties

If multiple objects of the same type are selected together their common properties will be shown in the properties tab.

This allows users to change the properties of hundresd of objects in a single click.

Using the Scene Browser to find problem assets

You can use the Scene Browser to locate problem assets by sorting the columns by Primitives or Triangles.
